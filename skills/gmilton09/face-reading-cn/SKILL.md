## 📖 技能说明

面相学是中国传统文化的重要组成部分，历史悠久，源远流长。本技能提供系统的面相学入门知识，帮助用户了解这一传统文化。

### 核心价值
- 📚 **文化传承** - 学习传统面相学知识
- 🔍 **快速查询** - 查询面部特征含义
- 🎯 **组合分析** - 多特征综合分析
- 🎭 **娱乐参考** - 趣味性的自我了解

### 基础理论（4 个）
1. **三停五眼** - 面部比例基础

3. **五行面相** - 金木水火土五行分类
4. **气色理论** - 面色与运势

### 五官详解（5 个）
1. **眉毛** - 兄弟宫、情感表达
2. **眼睛** - 监察官、心灵窗户
3. **鼻子** - 财帛宫、财富象征
4. **嘴巴** - 出纳官、表达能力
5. **耳朵** - 采听官、福气象征

### 面部区域（4 个）
1. **额头** - 事业宫、早年运势
2. **颧骨** - 权力欲、社交能力
3. **下巴** - 晚年运、稳定性
4. **法令纹** - 权威、社会地位

### 组合分析（4 个）
1. **富贵相** - 财富与地位特征
2. **事业运** - 职业发展特征

4. **健康运** - 身体健康特征

## 🚀 使用方式

### 方式 1: 直接查阅

```bash
# 进入技能目录
cd skills/face-reading-cn/

# 查看总导航
cat README.md

# 查看特定内容
cat BASICS/三停五眼.md
cat FEATURES/鼻子.md
```

### 方式 2: 快速查询

```bash
# 使用查询工具
python3 scripts/analyze.py "鼻子高挺"
python3 scripts/analyze.py "眉毛浓密 眼睛大"
```

### 方式 3: 完整分析

```bash
# 使用分析模板
python3 scripts/analyze.py --template "完整面相分析"
```

### 方式 4: 触发式调用

在对话中直接触发：
- "面相学中鼻子代表什么？"
- "什么是三停五眼？"
- "分析我的面相特征"
- "/face-reading 眉毛浓密说明什么"
