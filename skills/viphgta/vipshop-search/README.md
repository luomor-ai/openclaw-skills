# Vipshop Product Search Skill

唯品会商品搜索助手，根据关键词搜索商品并获取详细信息，包括商品名称、价格、品牌等关键信息。

## 功能特性

- 📦 **关键词搜索**：支持输入任意商品关键词进行搜索
- 🔄 **URL自动编码**：自动对搜索关键词进行URL编码处理
- 📊 **批量获取**：单次请求最多返回20个商品的详细信息
- 🎯 **详细信息**：返回商品ID、名称、价格、原价、折扣、卖点、品牌、商品链接等完整信息
- 📋 **结构化输出**：脚本输出JSON格式数据，LLM格式化展示给用户
- 🌐 **多环境适配**：自动识别用户环境并输出相应格式（企业微信、QQ、飞书、钉钉、微信、Web）
- ⚠️ **异常处理**：完善的错误处理和用户提示机制
- 🔄 **降级方案**：如果详情接口失败，自动降级为仅显示商品ID
- 🔑 **登录态支持**：自动读取 vipshop-user-login 的登录态，mars_cid 作为参数，PASSPORT_ACCESS_TOKEN 通过 cookie 传入

## 使用场景

- 🔍 需要查找特定商品的详细信息
- 📝 批量获取商品列表用于数据分析
- 🛒 商品价格比较和筛选
- 📊 市场调研和竞品分析
- 💰 了解商品的品牌、价格、销量等信息

## 接口信息

**登录态要求**：
- 使用搜索功能前，用户必须先通过 `vipshop-user-login` skill 登录
- `mars_cid` 作为请求参数从登录态文件读取
- `PASSPORT_ACCESS_TOKEN` 通过 Cookie 方式传入
- 如果未登录，系统会提示用户先完成登录

### 搜索接口

**请求URL：**
```
https://mapi-pc.vip.com/vips-mobile/rest/shopping/pc/search/product/rank
```

**请求方式：**
- GET

**主要参数：**
- `keyword`：搜索关键词（自动URL编码）
- `mars_cid`：设备ID（从登录态读取或使用默认值）
- `pageOffset`：分页偏移量（当前为0）
- `batchSize`：每页返回数量（固定为20）
- `channelId`：渠道ID（固定为1）

**Cookie：**
- `PASSPORT_ACCESS_TOKEN`：访问令牌（从登录态读取，如果已登录）

**返回数据：**
- `code`：状态码（1表示成功）
- `data.total`：匹配的商品总数
- `data.products`：商品列表，每个商品包含`pid`字段（注意：搜索接口返回的是`pid`，但详情接口使用的是`productId`）

### 商品详情接口

**请求URL：**
```
https://mapi-pc.vip.com/vips-mobile/rest/shopping/pc/product/module/list/v2
```

**请求方式：**
- GET

**主要参数：**
- `productIds`：商品ID列表（逗号分隔）
- `scene`：场景标识（固定为search）
- `mars_cid`：设备ID（从登录态读取或使用默认值）
- `user_id`：用户ID（从登录态读取或使用默认值）
- 其他参数已固化

**Cookie：**
- `PASSPORT_ACCESS_TOKEN`：访问令牌（从登录态读取，如果已登录）

**返回数据：**
- `code`：状态码（1表示成功）
- `data.products`：商品详情列表，包含：
  - `productId`：商品ID（注意：与搜索接口的`pid`不同）
  - `title`：商品名称
  - `price.salePrice`：当前售价
  - `price.marketPrice`：市场价
  - `price.saleDiscount`：折扣（如"1.7折"）
  - `price.priceLabel`：价格标签（如"特卖价"）
  - `price.sellTips`：销售提示（如"30天低价"）
  - `brandShowName`：品牌名称
  - `smallImage`：商品图片URL（小图）
  - `squareImage`：商品图片URL（方形图）
  - `attrs`：商品属性数组（包含name和value）

🏷️ **商品名**：商品的完整名称（title字段）
💰 **价格**：当前售价（price.salePrice字段）
💰 **原价**：市场价（price.marketPrice字段）

🏷️ **品牌**：商品所属品牌（brandShowName字段）

*注：原价、折扣、卖点和品牌字段可能为空，如果为空则不显示对应行*

## 注意事项

1. **登录要求**：使用搜索功能前必须先通过 `vipshop-user-login` skill 登录，未登录时会提示用户
2. **单次限制**：当前每次最多返回20个商品的详细信息（为了保证详情获取的准确性）
3. **参数固化**：大部分接口参数已固化，无需手动调整
4. **网络要求**：需要正常的网络连接才能调用接口
5. **接口更新**：如果接口参数失效，会提示接口可能已更新
6. **URL编码**：所有中文和特殊字符会自动进行URL编码
7. **字段缺失**：如果某些字段为空，会自动跳过该字段的显示
8. **字段名称差异**：
   - 搜索接口返回的`pid`字段值与详情接口的`productId`字段值相同
   - 搜索接口和详情接口的字段名称不同（如`title` vs `productTitle`）
   - 详情接口没有销售数量字段
9. **价格信息**：包含当前售价、市场价、折扣、销售提示等多个维度
10. **登录态**：脚本会自动读取 `~/.vipshop-user-login/tokens.json` 文件，如果用户未登录，会提示用户使用 `vipshop-user-login` skill 进行登录
11. **多环境输出**：系统会自动识别用户所在环境（企业微信、QQ、飞书、钉钉、微信、Web），并输出相应格式的内容

## 工作流程

```
用户输入关键词
    ↓
检查登录态（~/.vipshop-user-login/tokens.json）
    ↓
如果未登录：
    ↓
    检查 vipshop-user-login skill 是否已安装
    ↓
    如果已安装：引导用户使用 vipshop-user-login skill 登录
    ↓
    如果未安装：提示用户先安装 vipshop-user-login skill
    ↓
    等待用户完成登录
    ↓
如果已登录：
    ↓
执行 Python 脚本：python3 scripts/search.py <关键词>
    ↓
脚本读取登录态（mars_cid 和 PASSPORT_ACCESS_TOKEN）
    ↓
搜索接口获取商品ID列表
    ↓
提取所有商品ID
    ↓
调用商品详情接口
    ↓
脚本输出 JSON 格式数据
    ↓
LLM 解析 JSON 并格式化展示给用户
```

## 常见问题

**Q: 为什么只返回20个商品的详细信息？**
A: 当前设置batchSize=20，这是为了保证商品详情获取的准确性和响应速度。未来可扩展分页功能获取更多商品。

**Q: 支持哪些语言的搜索？**
A: 支持中文、英文等多种语言的关键词搜索，会自动进行URL编码处理。

**Q: 接口调用失败怎么办？**
A: 检查网络连接是否正常，如果持续失败可能是接口参数已更新，请联系管理员。如果详情接口失败，系统会自动降级为仅显示商品ID。

**Q: 商品ID可以做什么用？**
A: 商品ID是唯品会商品的唯一标识，可以用于商品详情查询、购物车添加、订单处理等多种场景。

**Q: 为什么有些商品没有显示所有字段？**
A: 某些商品可能缺少品牌、原价、折扣、销售提示等信息，系统会自动跳过空字段，只显示有值的信息。

**Q: 搜索接口返回的pid和详情接口的productId有什么区别？**
A: 它们的值是相同的，只是字段名称不同。搜索接口使用`pid`字段，详情接口使用`productId`字段，但这两个字段的值完全一样，可以互换使用。

**Q: 为什么没有显示销售数量？**
A: 详情接口的返回数据中没有销售数量字段，所以无法显示。如果需要销量信息，可能需要调用其他接口。

**Q: 可以按价格筛选商品吗？**
A: 当前版本暂不支持价格筛选，但可以在获取结果后根据价格信息进行手动筛选。未来版本会增加此功能。

**Q: 是否需要登录才能使用？**
A: 是的，必须先通过 `vipshop-user-login` skill 登录才能使用搜索功能。未登录时系统会提示您先完成登录。

**Q: 如何登录？**
A: 使用 `vipshop-user-login` skill 进行登录。您可以说"登录唯品会"或"唯品会登录"。如果您的系统中没有安装此 skill，系统会提示您先安装。

**Q: 如何查看当前是否已登录？**
A: 检查 `~/.vipshop-user-login/tokens.json` 文件是否存在。如果存在且格式正确，说明已登录，搜索功能会自动使用登录态。

**Q: 如果未安装 vipshop-user-login skill 怎么办？**
A: 当您尝试搜索但未登录时，如果系统检测到您未安装 `vipshop-user-login` skill，会提示您先安装该 skill。请按照提示完成安装，然后使用该 skill 进行唯品会登录。

**Q: 商品链接格式是什么？**
A: 商品链接格式为：https://detail.vip.com/detail-{brandId}-{productId}.html，其中 brandId 和 productId 都从接口返回的数据中获取。
