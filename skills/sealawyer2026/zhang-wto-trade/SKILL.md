# WTO贸易规则AI助手 🌐

> 专业WTO贸易律师AI助手 | 争端解决 | 贸易救济 | 合规审查 | 谈判支持

## 🎯 适用场景

- WTO争端解决（DSU）
- 反倾销/反补贴调查应对
- 保障措施调查
- 贸易政策合规审查
- 自由贸易协定（FTA）谈判
- 出口管制合规
- 经济制裁应对

## 📋 阶段一：WTO争端解决

### 1.1 WTO争端策略制定
```
请制定WTO争端解决策略：

争议背景：
- 争议措施：[措施描述]
- 涉及协定：[GATT/GATS/TRIPS等]
- 受影响成员：[成员方]

策略分析：
1. 磋商可行性
2. 专家组程序评估
3. 上诉程序评估
4. 执行程序评估
5. 报复授权评估
6. 时间成本分析
7. 政治经济考量
8. 最优路径建议
```

### 1.2 磋商请求起草
```
请起草WTO磋商请求：

请求方：[成员]
被请求方：[成员]
争议措施：[措施]
涉及协定：[协定]

磋商请求应包含：
1. 当事方信息
2. 争议措施描述
3. 涉及协定条款
4. 初步证据
5. 磋商请求
6. 法律依据
```

### 1.3 专家组申请
```
请起草专家组设立申请：

申请方：[成员]
被申请方：[成员]
磋商结果：[未解决/未回应]

申请应包含：
1. 当事方信息
2. 争议背景
3. 磋商情况
4. 专家组设立请求
5. 争议要点
6. 法律依据
```

### 1.4 专家组陈述
```
请起草专家组第一次书面陈述：

申诉方/被诉方：[身份]
争议焦点：[焦点]

陈述结构：
1. 事实背景
2. 法律主张
3. 证据支持
4. 对对方主张的回应
5. 结论请求
```

## 📋 阶段二：贸易救济应对

### 2.1 反倾销调查应对
```
请制定反倾销调查应对策略：

调查机关：[国别机构]
涉案产品：[产品描述]
调查期：[时间]

应对策略：
1. 市场经济地位申请
2. 分别税率申请
3. 抽样调查应对

8. 听证会准备
```

## 📋 阶段三：合规审查

### 3.1 贸易政策合规审查
```
请进行贸易政策WTO合规审查：

政策措施：[政策描述]
涉及领域：[货物/服务/知识产权]

审查要点：
1. 最惠国待遇（GATT I）
2. 国民待遇（GATT III）
3. 关税约束（GATT II）
4. 数量限制（GATT XI）
5. 非歧视原则
6. 透明度要求
7. 合规风险评估
8. 修改建议
```

### 3.2 出口管制合规
```
请进行出口管制合规审查：

目的地：[国家]
最终用户：[用户]

审查要点：
1. 管制清单审查
   - 中国管制清单
   - 美国EAR/ITAR
   - 欧盟管制清单
2. 最终用户审查
3. 最终用途审查
4. 许可证要求
5. 合规文件准备
6. 违规风险评估
```

