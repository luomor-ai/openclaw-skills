## 核心原理

### 一、太乙积年

太乙积年 = 从上元甲子至今的年数

**计算公式（简化）：**
- 基准年：公元前 2917 年（上元甲子）
- 积年 = 当前年份 + 2917

### 二、阴阳遁

| 时段 | 遁局 |
|------|------|
| 冬至 ~ 夏至 | 阳遁 |
| 夏至 ~ 冬至 | 阴遁 |

### 三、十六神将

| 神将 | 含义 |
|------|------|
| 太乙 | 主神，统领全局 |
| 文昌 | 文运、文化 |
| 计神 | 计谋、策略 |
| 始击 | 开始、攻击 |
| 主大 | 主方大将 |
| 客大 | 客方大将 |
| 主算 | 主方计算 |
| 客算 | 客方计算 |
| 定算 | 定局计算 |
| 四神 | 四方之神 |
| 天目 | 天文观测 |
| 地目 | 地理观测 |

### 四、主客算

- **主算：** 代表主方、内部、守势
- **客算：** 代表客方、外部、攻势
- **定算：** 综合判断

```

太乙：离宫

始击：震宫
主大：坎宫
客大：艮宫
...

【主客算】
主算：36（吉）
客算：24（平）

【大势判断】
2026 年太乙局显示：

```
