# Format

Code formatter — auto-format source code, consistent styling, language detection, batch formatting, config presets, and diff preview.

## Commands

| Command | Description |
|---------|-------------|
| `format run` | Execute main function |
| `format list` | List all items |
| `format add <item>` | Add new item |
| `format status` | Show current status |
| `format export <format>` | Export data |
| `format help` | Show help |

## Usage

```bash
# Show help
format help

# Quick start
format run
```

## Examples

```bash
# Run with defaults
format run

# Check status
format status

# Export results
format export json
```

- Run `format help` for all commands
format/`

## When to Use

- for batch processing format operations
- as part of a larger automation pipeline

## Output

Returns summaries to stdout. Redirect to a file with `format run > output.txt`.
