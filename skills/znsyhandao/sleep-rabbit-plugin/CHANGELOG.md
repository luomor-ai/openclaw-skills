# 眠小兔睡眠健康插件更新日志

## v1.0.3 (2026-03-22)

### 安全修复
- **移除`network`权限**: 从package.json中移除不必要的网络权限，减少攻击面
- **替换不安全的`exec`调用**: 将`child_process.exec`替换为更安全的`spawn`调用
- **修复依赖检查**: 使用安全的进程生成方式检查Python依赖
- **参数传递安全化**: 确保所有参数都安全传递，避免shell注入风险

### 功能改进
- **整合本地改进**: 将本地`plugins/`目录的更新文件整合到技能包中
- **更新插件文件**: 使用本地更新的`sleep_analyzer.py`等文件
- **版本统一**: 更新所有配置文件中的版本号为v1.0.3

### 代码优化
- **清理冗余代码**: 移除不必要的导入和代码片段
- **错误处理改进**: 增强错误处理和用户反馈
- **日志优化**: 改进控制台输出和错误信息

## 安全承诺

v1.0.3版本已解决以下安全问题：

1. **权限最小化**: 只保留必要的文件系统和进程权限
2. **进程安全**: 使用安全的`spawn`代替不安全的`exec`
3. **输入验证**: 所有用户输入都经过验证和清理
4. **本地处理**: 所有分析都在本地完成，不发送数据到外部服务器

## 安装要求

- **Python 3.8+**: 需要安装MNE, NumPy, SciPy等科学计算库
- **Node.js 16+**: 运行OpenClaw插件系统
- **系统权限**: 需要文件读取和进程生成权限

## 故障排除

如果遇到问题，请检查：

1. Python依赖是否安装: `pip install mne numpy scipy`
2. 文件权限是否正确
3. EDF文件格式是否支持
