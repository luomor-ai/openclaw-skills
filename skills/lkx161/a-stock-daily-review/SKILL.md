# A股每日收盘点评智能系统

> 每天晚上9点自动推送完整市场分析报告到微信，中国A股短线投资者必备工具。

## 功能模块

- **大盘概况**：上证/深证/创业板实时行情 + 市场情绪判断
- **热点板块深度分析**：结合近1-3个月数据 + 近3-5日突破信号
- **持仓盈亏跟踪**：实时股价/盈亏金额/盈亏比例
- **4只持仓股技术分析**：均线/RSSI/MACD/关键价位/⭐评分
- **1-2周主线板块预判**：逻辑强 + 调整充分 + 突破信号 + 催化因素
- **重点推荐核心股**：新标的（非持仓），完整技术分析
- **明日可执行操作计划**：精确价格/数量/金额/风险提示

## 选股方法论

1. 【数据窗口】分析近1-3个月数据
2. 【突破信号】近3-5日最重要，是板块启动核心信号
3. 【板块选择】逻辑强 + 近1-3个月有调整 + 近3-5日有突破迹象
4. 【推荐股票】新标的，非持仓股
5. 【推荐逻辑】板块逻辑 + 调整充分 + 突破逻辑 + 业绩 + 消息
