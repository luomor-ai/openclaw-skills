## Axure 使用方法

### 1. 添加内联框架
- 打开 Axure RP 11
- 拖入"内联框架"到画布
- 调整大小（建议 1200x800）

### 2. 粘贴代码
- 选中内联框架
- 交互 → 链接 → JavaScript
- 复制我生成的代码 → 粘贴 → 确定

### 3. 预览
- 按 F5 预览
- 内联框架显示可交互页面
