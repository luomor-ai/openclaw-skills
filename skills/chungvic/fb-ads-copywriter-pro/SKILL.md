**專業級 Facebook 廣告文案生成工具**

## 🎯 功能

- 📝 **6 個廣告版本** - FOMO/社交證明/權威/痛點/促銷/故事

## 🚀 快速開始

### 安裝
```bash
clawhub install fb-ads-copywriter-pro
```

### 配置
```bash
export GLM_API_KEY="your-glm-api-key"
```

### 使用
```bash
python scripts/copy-generator.py generate \
  --product "產品名稱" \
  --audience "目標受眾"
```

## 📋 6 個廣告風格

1. **FOMO** 🔥 - 限時優惠、庫存緊張
2. **社交證明** 👥 - 用戶評價、銷量
3. **權威背書** 👩‍⚕️ - 專家推薦、科學驗證
4. **痛點營銷** 💔 - 問題解決、情感共鳴
5. **優惠促銷** 🎁 - 首次購買、限時折扣
6. **故事營銷** 📖 - 品牌故事、用戶轉變
